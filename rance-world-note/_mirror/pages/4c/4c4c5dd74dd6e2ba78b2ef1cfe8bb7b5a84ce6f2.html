カスタムの町

ガイゼル・ゴード
年齢(生年)：56歳 / LP7年時点(GI966年生まれ)
レベル / 才能限界：LV1 / LV5
技能LV(傾向)：音楽LV1
カスタムの都市長、娘のチサを溺愛している。カスタム陥没以来ずっと病床生活。
当初は町の男性達による青年団が地下迷宮に入ったのだが、誰も戻ってこなかった。
冒険者ギルドに依頼を出しても、バードのパーティはマリアの迷宮にいる彫像相手にほぼ全滅、ミリはピラミッド迷宮にいるハニー相手に自分以外のメンバーが死亡、アリオスは依頼を受けながらも結局間に合わず、実に散々な結果だった。
ランスがキースから依頼を聞きカスタムに到達するも、予定の報酬20000GOLDから50000GOLDまでに強引に値上げさせられ渋々と承諾した。
またランス3のマニュアルによると、LP2年、都市長の任は既に娘のチサに受け継がれたが、鬼畜王ではチサが遠い町に嫁いだため代わりにエレノア・ランが都市長になった。
リメイクのランス03ではカスタム攻防戦の時、彼は依然都市長であるようで、ただ陥没した時の腰がまだ治ってないため、娘のチサがずっと傍で介護している。
チサ・ゴード
年齢(生年)：25歳 / LP7年時点(GI997年生まれ)
レベル / 才能限界：LV1 / LV3
技能LV(傾向)：音楽LV1
カスタムの都市長の娘。02でランスの初見評価は82点。
一時期行方不明になるが、ランス達に保護されて、マリアや志津香が解放された後、ランスを誘惑して肉体関係を持つ。
しかし、実はラギシスに身体を乗っ取られており、ランスの手に入れた4つの指輪を奪い取る。
ランス3のマニュアルによると、LP2年、彼女は既に父親のガイゼルから都市長の任が受け継いだが、鬼畜王では遠い町に嫁いだため代わりにエレノア・ランが都市長になった。
リメイクのランス03ではカスタム攻防戦の時、ガイゼルが依然都市長であるようで、ただ陥没した時の腰がまだ治ってないため、彼女はずっと傍で介護している。
結婚願望は人並みにあるのだが、父が候補者を次々に粉砕する。

エレナ・エルアール
年齢(生年)：26歳 / LP7年時点(GI996年生まれ)
レベル / 才能限界：LV1 / LV3
技能LV(傾向)：料理LV1
酒場の看板娘。
昔に参加した覆面社交パーティでナニに666の黒い数字が入っている男と出会い、一夜を共にした。それが彼女の初恋だった。
ランスがカスタムの酒場で彼女を口説くと、500GOLDを払うとHしてもいいと言い、しかし情事が終わった後、ランスも彼女の探している初恋の相手ではないことが判明した。ちなみにランスは最初の売春の相手だった。
その後、2、3人に抱かれた後、ついに666の男を見つけた。しかし付き合い出すとすぐに偽者と分かる。なんと彼はエレナが好きで、その噂を聞いて自らナニに666を彫ってエレナに近づいたのだった。
そして偽者と別れたエレナは、今も666のナニを持つ男を捜している。*3
ランス03では町の皆と一緒にヘルマン軍の攻撃に備えて疎開していた。
鈴木君
酒場のバーテン。ハニー。
せっかくアイテム屋から買った蒸発の実をうっかりうろろーんの中に落とした。
長柄亮子
役所にいる水色のポニーテールの女の子、ランスの変な噂を流している。
実は旧作でもエレノアの後ろにいる。苗字は02からつけられた。
ランス03のランスの台詞に出てくる「亮子ちゃん」は彼女のことであり、オーロラこと等々力亮子ではない。
ランス10の序盤の世界各国要人の会議ではエレノア・ランの隣に座っている。*4

カスタムの地下迷宮

牧場野ANT?
ALICEの館を探す老齢の戦士。
不運にも尋ねたのがランスであったため147GOLD巻き上げられた。
パソコン通信の参加ユーザーのハンドルネームを作中キャラとして登場させるという企画由来。
ランス・クエストで姓が設定された。
ランス・クエストではゼスにあるメグ牧場内の建物の瓦礫に埋まっていた。
なんと彼こそがメグ牧場の主、牧場野メグの父。
実の娘にも関わらず冒険から帰ってくると、その豊満なおっぱいに顔を埋めているとか。~
ネイ・ウーロン?
年齢(生年)：26歳 / LP7年時点(GI996年生まれ)
レベル / 才能限界：LV12 / LV13
技能LV(傾向)：盗賊LV1
シーフでバードのパーティーメンバー。当時はバードと付き合っていた。
マリアの迷宮にいる彫像に負けて満身創痍の状態でランスに見つかったが、そこのランスの行動選択によって彼女も違う反応をする。
一つはランスがその場で彼女を襲うと後に彼女からニンジンの鍵の情報を聞きだせる。
もう一つはシィルに彼女にヒーリングをかけさせるだけ、その後彼女はカスタムの酒場に出入りようになり、カエルの耳飾を持っていくとニンジンの鍵の情報を教えてくれる。
そしてこの件で限界を感じた彼女はその後結婚してしまい、今は普通の主婦をしている。
ペペ・ウィジーマ
年齢(生年)：22歳 / LP7年時点(GI1000年生まれ)
レベル / 才能限界：LV1 / LV2
技能LV(傾向)：
写真家、向けると笑顔でピースをしてしまう不思議なカメラを持っている。
最初はHな写真をランスがピラミッド迷宮で入手した石盤と交換したが、後にランスに反故され石盤を奪い返され涙目で走り去った。
ちなみにランス6の『ゼス美少女名鑑ＬＰ４年』の写真も彼女の作品である。
バ・デロス・ガイアロード?
年齢(生年)：
レベル / 才能限界：LV10 / LV10
技能LV(傾向)：統率LV1
ピラミッドの迷宮にいるミイラ、生前はリンゲル王ザーハードス６世に仕えた親衛隊副長。王の死の際に一緒に埋められた。リスガードルと言う名の娘がいた。
ミルの幻獣を倒せる幻獣の剣の情報以外、ランス達が酒場のうはあんと言う料理を持って行くと最後のワープコードを教えてくれる。
ランス10の時点でも依然変わらぬ状態でおり、自由都市魔人退治1でカスタムルートを選択すると、マップ上のオブジェクトとして確認できる。
シィルフィード
年齢(生年)：享年33歳
レベル / 才能限界：LV16 / LV16
技能LV(傾向)：槍LV1
志津香の部下の風の戦士(笑)。ラルガとの戦いに負け干からびており*5、ランスには名前でなく「クズ」と呼ばれる。享年33歳。
ちなみにラルガから鍵を貰って志津香の屋敷に入ると彼の仲間の他の風の戦士は汎用敵として襲って来る。しかしこいつらどういう経緯で志津香の部下になったのかは実に謎である。
ランス6で登場する同名のシィル・プラインの武器とは恐らく関係ない。
パソコン通信の参加ユーザーのハンドルネームを作中キャラとして登場させるという企画由来。

レベル / 才能限界：LV1 / LV8

今は学校の先生をしているが、たまに当時の事を想いだし時々エロい表情するので生徒には人気です。
