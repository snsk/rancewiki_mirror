カサドの町

よっちゃん
年齢(生年)：29歳 / LP6年時点(GI992年生まれ)
レベル / 才能限界：LV1 / LV6
技能LV(傾向)：
アイテム屋「よっちゃんの店」の店員、挨拶の言葉は「あんころぴー 」の変な女の子。「よっちゃん」自体が名前で、ちゃん付けだと「よっちゃんちゃん」、さん付けだと「よっちゃんさん」になるので間違わないように。ちなみにランスの彼女に対する評価は98点。
商品の値段は彼女の好感度によって決める、最大は定価の8倍、最小は定価の10分の1、彼女の願いや依頼を叶えるとだんだん安くなる、逆に彼女に意地悪すると膨れ上がるので注意。
浮力の杖でオーナーのレーガンの大好きなチェルネン粉を入手して店に持って行くと彼女付きで店自体を購入できる、商品を無料で取り放題以外、彼女を襲っても好感度が下がらない。
レーガン
「ぱぶぅ」としゃべる謎の生き物。よっちゃんの店の元オーナー、カサドの町の一番の金持ちらしい。
チェルネン粉が大好き。
ベル
年齢(生年)：27歳 / LP6年時点(GI994年生まれ)
レベル / 才能限界：LV2 / LV10
技能LV(傾向)：家事LV1
GI1015年に結婚した童顔の人妻、にとべくんという動物を飼っている。
町で出会った以降、ランスに狙われるようになる。後に町で会う度にランスにゴロツキのような因縁を付けられるが、パーティーにリックとレイラなどの騎士がいると邪魔される。そして彼らがパーティーから外されると、ベルもランスに人気のない所に連れられて犯されてしまう。おまけに青年団のメンバーが情けなさ過ぎる為に、結局この事件は揉み消しにされる等、つくづく不運であった。
にとべくん
ベルのペット、鳴き声は「だーーーだーーーだーーー」の謎の生き物。100歳以上らしい。
ベッキー
年齢(生年)：19歳 / LP6年時点(GI1002年生まれ)
レベル / 才能限界：LV12 / LV22
技能LV(傾向)：物書きLV1、音楽LV1
友人達よりもいち早く処女を捨てたいと嘯く、おませな女の子。
青年団のキセダがベル以外の被害者を出さない為に、彼女におつかい(?)でランスの遊び(SEX)相手を務めるように命じた。
幼く見えるが、実はアリシア、シンシア姉妹よりも1歳年上である事が後に判明しており、年齢的にはランスの許容範囲内であった。それを一切知らず、幼い声で誘ってくる少女というシチュエーションに惹かれたランスは外見が許容範囲外であるにもかかわらず、彼女を美味しく頂いてしまった。
イラーピュ落下後は小説家になっている。
マイ
年齢(生年)：20歳 / LP6年時点(GI1001年生まれ)
レベル / 才能限界：LV1 / LV7
技能LV(傾向)：
おかゆフィーバーの生贄にされそうだった女の子。行方不明になっていたアリシアとは親友関係にあった。
青年団に彼氏がいる非処女だったが、生贄になる事が決まった日に、彼氏からあっさり別れを持ち出されてしまっている。おかゆフィーバー撃退後、一度はランスとの約束のHを反故しそうになったが、既にパープルにも約束を反故にされたランスから逃げる事は出来ず、約束どおりのHをした。後に、救出されたアリシアの台詞から、マイがランスのテクニックに本気で感じていた事が判明。別れた彼氏の時は、ただ痛いだけであったが、Hの経験豊富なランスの責めはとてつもなくうまかったらしく、一晩で何回イッたか分からなかったらしい…。この事が、アリシアがランスに強く興味を抱く事になっている。
イラーピュ落下後は、おかゆフィーバーに関する被害者団体会員になっている模様。
パープル・ソウル
年齢(生年)：21歳 / LP6年時点(GI1000年生まれ)
レベル / 才能限界：LV1 / LV4
技能LV(傾向)：
GI1014年におかゆフィーバーの生け贄に捧げられた女の子。
裸の状態で牢屋に閉じ込められ、一度も挿入されずにただイカされるだけという欲求不満な思いをしていた。ランスが助ける代わりにHをする約束をしていたが、解放された後、ランスを突き飛ばして約束を反故にした。本来は恥ずかしがり屋な性格であったらしいが、おかゆフィーバーに責められ続けた結果、おかゆ毒に侵されてしまい、それによる狂淫作用から、ランスを誘惑してHする事になる。後にその真実が神官のシンシアによって明かされ、ランスに「フィーバー下し」で治療された結果、元に戻るが、恥ずかしさのあまり悲鳴をあげて逃げ出した。
イラーピュ落下後は、おかゆフィーバーに関する被害者団体会員になっている模様。
キセダ・エスピラ
年齢(生年)：享年25歳
レベル / 才能限界：LV1 / LV1
技能LV(傾向)：
青年団団長。ガリガリのヘタレ。
青年団のリーダーでありながら頼り甲斐の無い人物で、生贄にされそうになったマイを助けようとしたランスの行為に反対したり、脅されて青年団の運営費用をあっさり渡してしまう等、典型的な保守派。後に人妻のベルがランスに強姦される事件に遭遇し、さすがにこの時はランスを逮捕しようとしたが、生贄の件の負い目や彼が街で最強だったフロンソワーズ以上の実力者であった事から、結局は強引に事件の揉み消しを言い渡されてしまった。強姦事件に関しては、青年団でも問題視され、ランスに強姦を繰り返してほしくなかったら女を用意しろと脅された結果、ベッキーにおつかいを言い渡した。
ストーリー後の動向は全くの不明であったが、後に「ランスワールドノート」にて、イラーピュの落下時に死亡した事が明かされている。

闘神都市(迷宮)内

ナチュリ・ドラ
研究コアのB2Fにいた幽霊の女の子。生前はルクラ村に住んでいた。
イラーピュがまだ闘神都市として稼動していた頃、生体実験をされていた。偶然訪ねてきたランスに自分を成仏させてほしいと頼み、お礼に研究室の鍵を渡してくれる。
後にランスが同じB2Fの別室に彼女の骸骨を発見し、更に教会に行ってシンシアから聖水をもらって骸骨にかけるとやっと彼女を成仏させる。だが成仏する前に、最後の未練としてセックスしてみたいとランス達に言い、肉体がないならシィルの身体に乗り移ればいいとイオに勧められた。イオに操られたランスはそれを許可すると、ランスに逆らえないシィルも渋々と自分の身体を貸し出してナチュリに楽しい一時の思い出を残させた。
ハッピー福田
ボロボロの服を着たがりがりのオタク、上部中央エリアの一室に住んでいた。
よっちゃんの店に「ぴんてる486」を通信販売で注文した。
テッド
闘将コアのB5Fの一室で磔にされていた男。
ランスが自分の主魔人レキシントンの後継者に相応しいかどうかをテストするために、アトランタが用意した。
ナナミ
闘将コアのB5Fの一室で鎖で壁に繋がれていた全裸の女。
ランスが自分の主魔人レキシントンの後継者に相応しいかどうかをテストするために、アトランタが用意した。
ナゾのガキ
上部中央エリアにいた女の子、ランス的にはあと5年したら守備範囲に入る。
ランスが東の部屋にいる山のサチと海のサチを全部襲えたらPTTP薬をくれる。それを夜伽モードでシィルに使うとちょっとした特殊なプレイが出来る。
